This is the development file.

Ether got included with the following line.

<script src="../js/ether.js" data-main="scripts/start.js"></script>

After setting up the Ether namespace start.js get included and will be run. You should see some alerts-Popup.

To now create the release code just run ether.exe from the commandline with the following command:

ether.exe example example/scripts/start.js example/scripts/release.js

There is already one prepared for you. You can just watch it's output in release.html.
